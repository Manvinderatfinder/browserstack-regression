# regression-suite
## Installation
```shell
npm install webdriverio
```

## Testing
- Test coverage per niche can be found [here].
#### Run the regression-suite using the following command:
- Suite includes tests for Money, Global User, Shopping, Tech & Telco and Insurance
```shell
npm run regression
```
#### Run per niche using the following command:
Money
```shell
npm run mny
```
Global User
```shell
npm run global
```
Shopping, Tech & Telco
```shell
npm run ste
```
Insurance
```shell
npm run ins
```
Finder US
```shell
npm run finus
```
Mobile
```shell
npm run mobile
```
